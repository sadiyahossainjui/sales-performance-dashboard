# Sales Performance Dashboard

## Project Overview

This project is an interactive Tableau dashboard created using the Superstore dataset.

The dashboard analyzes:

* Sales performance
* Profit distribution
* Top-performing cities
* Geographic sales trends
* Category-wise sales analysis

## Tools Used

* Tableau
* CSV Dataset
* Data Visualization

## Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Profit Share by Region
* Top 5 Cities Analysis
* Sales by Category
* Sales by State Map
* Interactive Dashboard Filters

## Key Insights

* Technology generated the highest sales.
* California showed the strongest sales performance.
* New York City was the top-performing city.
* Furniture had lower profit margins compared to other categories.
